# Projeto-Afya-2up
Sistema de Agendamento e Telemedicina para Clínicas Populares O Problema: Agendar consultas médicas presenciais ou online, gerenciar salas/links de videoconferência, prontuários eletrônicos e receitas médicas digitais. Complexidade DDD/OO: Contextos de "Agenda Médica", "Atendimento/Prontuário" e "Faturamento".
